# PlanLoop · 计划

免费、极简、**循环任务一目了然、不制造打卡焦虑**的手机计划工具。纯前端 PWA，数据存在本机，无需注册、无服务器、零成本。

> 当前版本：**v1（本地版）**。完整设计与路线图见 [`docs/2026-06-21-planloop-design.md`](docs/2026-06-21-planloop-design.md)。

## 功能

- **三个页面**：今天（打卡）/ 计划（月历 + 任务管理）/ 我的（统计与数据）。
- **任务类型**：
  - 定性 —— 完成即可（读书、写日记）。
  - 定量 —— 带单位与目标，可记录真实值；支持「越多越好 ≥ / 越少越好 ≤」；可设为一天内多次累加（喝水：点 + 展开「一口/几口/半杯/一杯/自定义」，带毫升提示）。
- **循环类型**：每天 / 每周固定几天 / 每周 N 次 / 每月 N 次 / 一次性。
  - 「每周·每月 N 次」会每天提醒，达成 N 次后自动收进「已完成」，下周期重置。
- **月历视图**：计划页打开即月历，每天用颜色深浅表示完成度，点某天可查看 / 补打卡。
- **时段屏蔽**：支持「一段日期（一次性）/ 每周固定几天 / 每月固定日期段（循环）」三种，按理由（旅行 / 受伤 / 放假 / 自定义、颜色区分）屏蔽；也可在单个任务里设休息时段（如锻炼任务的每月经期休息）。屏蔽期间任务不出现、不计入达成率、不打断配额——名副其实的「放假」，不制造亏欠感。
- **左滑快捷操作**：今天页左滑任意任务行，可「今天休息」（仅今天暂停、不计入统计，明天自动回来）或「删除」整个任务。
- **任务详情图表**：在「计划」或「我的」点开任意任务，看本周柱状图（定量，带目标虚线）或坚持热力图 + 本周圆点（定性，绿=完成，不显示连续天数）。
- **智能模板 + 个性化**：新建任务可从模板（喝水 / 睡眠 / 走路 / 跑步 / 冥想…）快速开始；填了「我的→个人资料」后，喝水按 EFSA(2010)/体重法、睡眠按年龄（National Sleep Foundation）推荐个性化每日目标，小灰字标注来源。
- **温柔反馈（低压力）**：完成任务给随机鼓励，今日全部做完有特别一句；「我的」页有“本周回顾”正向小结——只夸努力、不提缺口、不显示连续天数。
- **随手记（记录型任务）**：不算达成率、不制造压力，只记录看趋势。今天页单独成区；内置「起飞 🚀」（趣味计数，点 + 记一笔）与「睡眠」（填上床/起床自动算时长，带趋势图与参考线）。从右上角 ＋ 的模板里添加。
- **统计**：顶部本周完成数 + 本周达成率；单任务本周进度；累计完成总数（藏在「我的」页，不在主页制造压力）。
- **数据**：导出 / 导入 JSON 备份；手动清理旧记录（累计数保留）；可一键重置演示数据。
- **PWA**：可「添加到主屏幕」像 app 一样使用，支持离线。

> 首次打开预置 3 个起步任务（喝水 / 读书 / 晨跑）和一张「三步上手」提示卡，无打卡历史。可在「我的 → 清空全部数据」重置回此状态。

## 本地运行

PWA 的 Service Worker 需要通过 http 访问（不能直接双击 `file://` 打开）。在项目目录起一个静态服务器即可：

```bash
# 方式一：Python（已安装）
python -m http.server 5173
# 然后浏览器打开 http://localhost:5173

# 方式二：Node
npx serve .
```

> 直接双击 `index.html` 也能用，只是离线缓存不生效，不影响功能体验。

## 在手机上用 / 分享给朋友

1. 把这个文件夹部署到任意静态托管（GitHub Pages、Vercel、Netlify、Cloudflare Pages 均可免费）。
2. 把链接发给朋友，手机浏览器打开。
3. iOS：分享 → 添加到主屏幕；Android：菜单 → 安装应用。即可像原生 app 一样使用。

## 之后做 App Store / 安卓上架

同一套代码可用 [Capacitor](https://capacitorjs.com/) 打包成原生壳子提交商店，无需重写。需要 Apple 开发者账号（$99/年）并过审核。

## 目录结构

```
PlanLoop/
├─ index.html            页面骨架 + 底部导航
├─ manifest.webmanifest  PWA 清单
├─ service-worker.js     离线缓存
├─ css/styles.css        样式
├─ js/
│  ├─ model.js           纯逻辑：日期/循环/完成判定/统计（可 node 自测）
│  ├─ storage.js         数据层：localStorage + 所有写操作 + 备份/清理
│  └─ app.js             界面：渲染 + 交互 + 模态框
├─ icons/                应用图标
├─ tests/model.test.js   逻辑自测（node tests/model.test.js）
└─ docs/                 产品设计文档
```

## 路线图

- **v2（已完成）**：计划页月历视图 + 完成度热力；时段屏蔽（全局 + 单任务，放假 / 受伤 / 经期休息等）。
- **v2.x（已完成）**：任务详情可视化——点开任务看定量柱状图（含目标线）/ 定性坚持热力图 + 本周圆点。
- **v3（已完成）**：个人资料（性别/体重/年龄/运动量，可选）+ 智能模板库；喝水按 EFSA(2010) 与 30–35ml·kg⁻¹ 体重法个性化推荐每日目标，小灰字标注来源。
- **以后**：云同步（换设备数据跟随）、好友 / 分享、提醒通知、上架 App Store。

## 设计取舍

- 数据**仅存本机浏览器**：换设备、卸载或清空浏览器数据会丢失 —— 请定期「导出备份」。云同步留到后续版本（数据层已封装，便于平滑升级）。
